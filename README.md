**SQL E-Commerce Sales Analytics with Power BI**

**Project Overview**

This project demonstrates an end-to-end E-Commerce Sales Analytics solution using SQL for data analysis and Power BI for interactive dashboard development.

The project focuses on extracting business insights from sales data through advanced SQL queries and visualizing key performance indicators in Power BI.

**Technologies Used**

-SQL (MySQL)

-Power BI

-CSV Dataset

-GitHub


**Project Features**

Database creation and management

Data analysis using SQL

Joins and aggregations

Views

Business insight queries

KPI dashboard

Sales performance analysis

Customer and payment analysis

Interactive Power BI dashboard with slicers
